# 🎮 Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript. Play with your friends and enjoy the classic X vs O game with a clean and responsive interface.

## 🚀 Live Demo

🔗 https://tic-tac-toe-xol3.onrender.com

## ✨ Features

- Two-player gameplay
- Winner detection
- Draw detection
- Reset game functionality
- Responsive design
- Simple and user-friendly interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```
├── index.html
├── style.css

```
